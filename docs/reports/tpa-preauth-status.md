# TPA Preauth Status Report

## Purpose

Provides a filterable overview of all TPA pre-authorization requests across the facility.

## Type

Script Report

## Reference DocType

TPA Preauth Request

## Filters

- Status
- Payer Type
- Insurance Payor
- Treating Practitioner
- Company
- From Date / To Date

## Columns

ID, Patient, IP Record, Payer Type, Insurance Payor, Status, Requested Amount, Approved Amount, Reference Number, Practitioner, Diagnosis, Submitted On, Approved On, Valid From, Valid To

## Access Roles

- Healthcare Administrator
- TPA Desk User
- Accounts User
