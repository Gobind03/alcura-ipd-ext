# TPA Claim Pack Status Report

## Purpose

Provides a filterable overview of all TPA claim packs for tracking submission and settlement progress.

## Type

Script Report

## Reference DocType

TPA Claim Pack

## Filters

- Status
- Insurance Payor
- Company
- From Date / To Date

## Columns

ID, Patient, IP Record, Insurance Payor, Status, Submitted Date, Reference, Settlement Amount, Settled On, Disallowed, Prepared By

## Access Roles

Healthcare Administrator, TPA Desk User, Accounts User
